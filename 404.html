Frequently Asked Questions/Learn more about this error message

Why am I getting this error page?

You are getting this error page because you typed the website name incorrectly. Parts of sites names are case sensitive but others are not. For example say if you were trying to access "example.com/test.html", it would fail if you typed it out like this: "example.com/TeSt.html" but it wouldn't fail if you typed it like this: "eXaMPle.CoM/test.html".
